# netsnmp

> Minimalistic SNMP client for Node.js

## INSTALLATION

### Requirements

- Node.js >= 8.0.0
- [net-snmp](http://www.net-snmp.org)

### Instructions

#### With `npm`

```bash
$ npm i netsnmp
```

#### With `yarn`

```bash
$ yarn add netsnmp
```
