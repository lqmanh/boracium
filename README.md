# BORACIUM

> Minimalistic SNMP client for Node.js

## Installation

### Requirements

- Node.js >= 8.0.0
- [net-snmp](http://www.net-snmp.org)

### Instructions

#### With `npm`

```bash
$ npm i @lqmanh/boracium
```

#### With `yarn`

```bash
$ yarn add @lqmanh/boracium
```
